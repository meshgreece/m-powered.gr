# Ξεκινώντας

Από το μηδέν στην πρώτη σου μετάδοση σε περίπου 30 λεπτά.

## Βήμα 1: Διάλεξε Hardware

Μία συσκευή αρκεί για αρχή. Θα προσθέσεις άλλες αν χρειαστεί.

### Heltec V3 ή V4 — €35-40

Η πιο δημοφιλής για αρχή. Μικρή OLED οθόνη, ενσωματωμένη κεραία, ESP32-S3 με LoRa chip SX1262.

### LILYGO T-Beam — €50-70

Έχει ενσωματωμένο GPS και θήκη για μπαταρία 18650. Καλή για κινητούς κόμβους που θέλουν αυτονομία.

### Seeed Studio SenseCAP T1000-E — €50-60

GPS tracker σε μέγεθος πιστωτικής κάρτας (85×55mm). Κρατάει μέρες με μία φόρτιση χάρη στον nRF52840. Βολικό για σακίδιο, κατοικίδιο ή όχημα.

### LILYGO T-LoRa-T3-S3 — €30-35

Η πιο φτηνή. Βασική πλακέτα ESP32-S3 χωρίς οθόνη ή GPS. Κάνει για σταθερό κόμβο ή DIY. **Προσοχή**: Παράγγειλε 868MHz (SX1262 ή SX1276), όχι 2.4GHz!

---

**Συχνότητα**: Όλες πρέπει να είναι 868MHz για Ελλάδα.

**Καλή αρχή**: Heltec V3/V4 — φτηνό, έχει οθόνη, και δοκιμάζεις το mesh πριν ρίξεις περισσότερα λεφτά.

## Βήμα 3: Εγκατάστησε το Firmware

Πηγαίνε στο [flasher.meshtastic.org](https://flasher.meshtastic.org) με Chrome ή Edge, σύνδεσε τη συσκευή με USB-C, επίλεξε το μοντέλο και πάτα Flash. Διαρκεί 2-3 λεπτά συνολικά.

## Βήμα 4: Διαμόρφωσε τον Κόμβο σου

Κατέβασε την εφαρμογή Meshtastic από το Play Store ή App Store, ή χρησιμοποίησε το [web client](https://client.meshtastic.org/) από υπολογιστή. Σύνδεσε στον κόμβο μέσω Bluetooth ή USB και άνοιξε τα **Settings**.

Τέσσερα ρυθμίσεις αρκούν για να ξεκινήσεις:

### 1. Περιοχή (Region)
**Πού**: Radio configuration > LoRa > Options > Region

**Τι να επιλέξεις**: **European Union 868Mhz**

Χωρίς αυτή τη ρύθμιση, δεν θα δεις κανέναν. Όλοι στην Ελλάδα χρησιμοποιούν 868MHz.

### 2. Όνομα Κόμβου
**Πού**: Device configuration > User

**Τι να κάνεις**: Βάλε ένα όνομα που να σε αναγνωρίζει (π.χ. "Γιάννης Αθήνα", "Alex_Thess", "Pilot42").

Το default "Meshtastic abcd" δεν λέει τίποτα σε κανέναν.

### 3. Ρόλος Συσκευής (Role)
**Πού**: Device configuration > Device

**Τι να επιλέξεις**: `CLIENT` (για κινητούς κόμβους και προσωπική χρήση)

⚠️ **Προσοχή**: Μην επιλέξεις `ROUTER` αν δεν ξέρεις γιατί το κάνεις! Περισσότερες πληροφορίες στη [Βασική Διαμόρφωση](basic-configuration.md).

### 4. Αριθμός Hops
**Πού**: Radio configuration > LoRa > Advanced > Number of Hops

**Τι να επιλέξεις**: **4 ή 5**

Το default είναι 3, αλλά για το Ελληνικό δίκτυο που είναι ακόμα αραιό, 4-5 hops βοηθούν τα μηνύματα να φτάσουν πιο μακριά από κόμβο σε κόμβο.

## Βήμα 5: Πες Γεια σου

Άνοιξε την εφαρμογή, πήγαινε στα Messages, στείλε ένα μήνυμα. 

Αν είσαι τυχερός και υπάρχουν κόμβοι στην περιοχή σου, θα πάρεις απάντηση. Αλλιώς έλεγξε τον χάρτη κάλυψης - το δίκτυο μεγαλώνει αλλά έχει ακόμα κενά.

## Βήμα 6: Ενεργοποίησε το MQTT

**Settings > LoRa**:
- **OK to MQTT**: Enabled
- **Ignore MQTT**: Disabled

Με το MQTT ενεργό, ο κόμβος σου επεκτείνει την εμβέλεια μέσω internet και εμφανίζεται σε εργαλεία παρακολούθησης του δικτύου όπως το [Meshview](https://meshview.m-powered.gr/) και το [Malla](https://malla.m-powered.gr/).

> **Σημείωση ιδιωτικότητας**: Η ρύθμιση **OK to MQTT** σέβεται την επιλογή σου. Αν την έχεις **απενεργοποιημένη**, τα μηνύματά σου και οι πληροφορίες του κόμβου σου **δεν** ανεβαίνουν στο internet μέσω MQTT.
